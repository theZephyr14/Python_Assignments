**Python for Data Science - Assignment 2**

**Overview**: This assignment includes simple Python exercises to practice basic programming skills, such as conditionals, loops, list comprehension, and function usage. Each exercise comes with examples and explanations.

**Exercises**:

1. **FizzBuzz Function**: A function to print numbers from 1 to *n*, substituting "Fizz" for multiples of 3, "Buzz" for multiples of 5, and "FizzBuzz" for multiples of both.
  
2. **Data Filtering**: Filter a mixed data list to extract only integers using list comprehension.

3. **To-Do List**: Create a basic to-do list with two functions:
   - `add_task(task)`: Adds a task.
   - `show_tasks()`: Displays all tasks.

4. **Temperature Converter**: A function to convert Celsius to Fahrenheit, applying a conversion formula and displaying results for a Celsius list.
